python challenge 7

http://www.pythonchallenge.com/pc/def/oxygen.html

这个题需要解析图片，之前没有接触到因此是查资料完成。
1.需要安装PIL库
2.截取黑白部分
3.将RGBA格式的像素每个像素值按照L8位黑白像素的格式转成一个acsii码值
4.将最后获得的答案转换为ASCII码

最后将[105, 110, 116, 101, 103, 114, 105, 116, 121]转为ascii即为答案:integrity
